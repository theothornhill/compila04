** Tools

*** Tools: JFlex                                                    :B_frame:
    :PROPERTIES:
    :BEAMER_env: frame
    :END:


- scanner generator (or lexer generator) tool

   - *input*: lexical specification
   - *output*: scanner program in Java

- lexical spec written as ~.lex~ file

- consists of *3 parts*

   - user code
   - options and macros
   - lexical rules
